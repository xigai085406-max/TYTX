# 简历优化 Agent

## 背景

求职者常常不知道如何针对目标岗位优化简历。请你构建一个 **可运行的简历优化 Agent 应用**，帮助用户分析简历与目标岗位的匹配度，并在用户确认后生成优化后的简历。

## 任务目标

请实现一个完整应用，至少支持以下能力：

1. 用户既能上传一份简历文件（word,pdf,md等格式），也可以直接粘贴简历文本
2. 用户输入目标职位描述（JD）
3. 系统分析简历与 JD 的匹配情况，并在原始简历提供的事实基础上输出：
   - 匹配亮点
   - 主要缺口
   - 具体优化建议
4. 用户确认后，系统生成一份优化后的简历内容
5. 用户既能直接查看简历内容，也可以导出优化后的简历为（word,pdf,md等格式）

## 运行与交付要求

1. 应用必须能够通过 Docker 方式构建和启动
2. 仓库根目录下必须包含 `Dockerfile`
3. 如有需要，可以额外提供 `docker-compose.yml`
4. 必须提供完整 `README.md`，清楚说明：
   - 如何配置环境变量
   - 如何构建镜像
   - 如何启动服务
   - 如何访问和使用应用
5. 面试官应能够仅根据 README 完成项目启动和基本验收
6. LLM API Key 必须通过环境变量配置，不允许硬编码在代码中
7. 请提交一个可公开访问的 GitHub 仓库

## 评估重点

我们会重点关注：

1. 功能是否完整可用，简历分析与优化结果是否有实际价值
2. 优化建议与优化结果是否忠于原始简历的事实
3. Agent架构,包括Harness/Context Engineering，Memory管理是否合理，健壮。
4. Docker 部署是否顺畅， README 是否清晰，代码结构是否合理
